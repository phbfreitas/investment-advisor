# MANUAL DE OPERAÇÕES: INVESTMENT ADVISOR (Gold Standard)

Este manual documenta a lógica matemática, a arquitetura de dados e as diretrizes operacionais do ecossistema Investment Advisor. Projetado para garantir a integridade absoluta do patrimônio, o sistema opera sob o princípio do **Ripple Effect** (Efeito Cascata), onde cada entrada de dado reverbera em toda a inteligência analítica.

---

# PILAR 1: Fundação & UX / Foundation & UX

## LIVRO I: Gênese & Arquitetura / Genesis & Architecture

### A Diretriz Primária / Primary Directive
A plataforma é organizada em dois pilares fundamentais de navegação: **My Blueprint** (seu cofre privado de dados estáticos e dinâmicos) e **Market Intelligence** (motores de orientação ativa e dados externos). Toda a arquitetura foi construída para que o usuário entenda exatamente como o software processa a informação.

*   **Portuguese (PT-BR):** Este guia foi construído para ajudar usuários a entenderem exatamente **como o software pensa**.
*   **English (EN):** This guide is built to help users understand exactly **how the software thinks**.

### Arquitetura Ripple / Ripple Architecture
O "Efeito Cascata da Verdade" é o coração do sistema. Uma alteração em um **Ticker** ou em um **Risk Tolerance Slider** gatilha mudanças instantâneas em todo o sistema. Se você alterar seu perfil de risco em **My Investment Strategy**, o **AI Guidance** instantaneamente recodifica seus conselhos.

> "Sua riqueza não é estática; ela é uma rede de funções matemáticas interconectadas."
> "Your wealth is not static; it is a web of interconnected mathematical functions."

## LIVRO II: Taxonomia da Interface / Interface Taxonomy

### Estratégia de Sidebar / Sidebar Strategy
A interface é dividida em dois domínios ontológicos: **My Blueprint** (estático/histórico) e **Market Intelligence** (dinâmico/futuro). Esta separação garante que o usuário saiba se está olhando para seus próprios dados ou para a análise do mercado.

### Elementos Globais / Global Elements
A barra superior contém o **Universal Search Oracle** e o seletor de **Household**. Cada elemento é projetado para acesso instantâneo, independentemente de onde você esteja na aplicação.

# PILAR 2: Governança / Governance

## LIVRO III: Guardião de Identidade / Identity Guardian

### Filosofia & Risco / Philosophy & Risk
O **Risk Tolerance Slider** (escala 1-10) é o volante da IA. Um valor de 1 é o perfil **Conservative**; 10 é totalmente **Speculative**. Este número calibra todos os conselhos gerados no sistema, alterando os circuitos de decisão dos consultores virtuais.

### Gestão de Drift / Drift Management
O sistema compara seus alvos (**Target %**) contra o portfólio real (**Actual %**) e exibe uma tabela de **Drift**. Desvios maiores que 5% disparam alertas visuais imediatos, exigindo rebalanceamento para manter a integridade da estratégia.

---

## LIVRO IV: Governança de Portfólio / Portfolio Governance

### Atomic Registry (Registro Atômico)
Cada **Holding** é um objeto matemático único. O sistema rastreia **Ticker**, **Quantity**, **Book Cost** (Preço Médio) e integra com dados do **Yahoo Finance** para calcular o **Market Value** e o **Unrealized Gain/Loss** em tempo real.

### Edição e Automação / Inline Editing & Automation
As edições de ativos ocorrem diretamente na tabela (Inline). A busca de tickers é automatizada e a interface usa indicadores de estado para confirmar que cada mudança foi capturada no **Audit Trail** universal.

---

## LIVRO V: Motor Time Machine / Time Machine Engine

### Audit Trail Ledger / Livro de Auditoria
Cada mutação do portfólio — manual ou via **PDF Import** — é registrada com snapshots de "antes" e "depois". O **Time Machine** permite visualizar **Diffs** coloridos para cada transação e entender a origem de cada centavo no sistema.

### Cascade Rollback
Permite reverter qualquer alteração e todas as subsequentes em ordem cronológica inversa. Isso garante a integridade dos dados, prevenindo registros órfãos ou estados inconsistentes durante rebobinagens históricas.

# PILAR 3: Inteligência & IA / Intelligence & AI

## LIVRO VI: Market Intelligence

### Global Radar (Sinal Macro)
O **Global News Guidance** conecta manchetes geopolíticas ao seu cofre. Ele utiliza integração com **NewsData.io** para realizar testes de estresse em tempo real, analisando como taxas de juros ou conflitos afetam seu **Net Worth**.

### Strategic Directives (O Motor Analítico)
Através de diretivas de IA (como **Rebalancing** ou **Dividend Optimization**), o sistema gera relatórios multi-página com dados em streaming. Inclui avisos de **Stale Data** caso seu portfólio mude antes de você atualizar a análise.

---

## LIVRO VII: Ingestão PDF / PDF Import

### Fluxo de 3 Estágios / 3-Stage Workflow
1.  **Parsing**: Extração de texto via OCR e processamento de linguagem natural.
2.  **Validation**: Verificação de duplicatas e limpeza de ruído estatístico.
3.  **Atomic Sync**: Gravação irreversível no banco de dados via **Time Machine**.

### Atomic Sync & Deduplication
O sistema resolve automaticamente **Tickers** canadenses (adicionando .TO) e cruza dados por **Ticker** + **Account ID**. Registros redundantes são podados automaticamente para manter o "Gold Standard" da verdade.

## LIVRO VIII: O Guru AI (RAG)

### Motor de Contexto / Context Motor
Utilizamos **RAG** (*Retrieval-Augmented Generation*) para injetar seus dados privados (**Holdings** e **Strategy**) no contexto da IA com segurança. Isso garante que o **Guru** nunca dê conselhos genéricos; ele sempre sabe exatamente o que você possui.

> "A IA não apenas conversa; ela audita sua realidade antes de falar."
> "The AI doesn't just chat; it audits your reality before speaking."

# PILAR 4: Finanças & Segurança / Wealth & Security

## LIVRO IX: Alquimia de Finanças / Cash Flow & Real Estate

### Budget vs Actuals (O Reflexo do Gasto)
Integramos seu orçamento mensal diretamente à sua capacidade de investimento. O sistema monitora o **Investable Cash** e sugere aportes baseados no excedente mensal, garantindo que o plano de aposentadoria seja executado.

### Rental Reality (Yield Imobiliário Real)
O motor imobiliário desconta automaticamente taxas de condomínio, impostos e manutenção para entregar o **Net Yield**. Compare seus imóveis com o retorno de **Dividends** do mercado financeiro instantaneamente.

## LIVRO X: Protocolos de Segurança / Fortress Security

### Identidade Atômica / Atomic Identity
Segregação completa de dados com isolamento criptográfico ao nível do **Household**. Seus ativos nunca se misturam aos de outros usuários, garantindo privacidade absoluta.

### Household Governance
Compartilhe seu cofre com stakeholders. Convide cônjuges ou planejadores para sua partição segura. O acesso é revogável e controlado por chaves de convite exclusivas.

# PILAR 5: Referência / Reference

## LIVRO XII: Troubleshooting

### Error Dictionary & Recovery
*   **PDF Parse Failure**: Verifique se o arquivo é um export oficial do banco (não utilize scans).
*   **Guru Timeout**: Simplifique a pergunta ou reduza o número de documentos selecionados no contexto.
*   **Sync Conflict**: Ocorre quando dados locais divergem do banco. Utilize o botão **Refresh** para sincronizar.

## LIVRO XIII: Filosofia / Philosophy

### Os 3 Pilares da Lógica / The 3 Pillars of Logic
1.  **Integridade**: Dados puros. O sistema recusa interpolações criativas. Se não está no cofre, a resposta é "Não sei".
2.  **Inteligência**: Análise preditiva vs Reativa. O Oráculo foca no que o ativo *será*.
3.  **Resiliência**: Proteção contra ruído. Baixa taxa de giro e foco no longo prazo.

## LIVRO XIV: Glossário do Oráculo / Terms & Definitions
*   **Beta**: Média de volatilidade relativa ao mercado (**Benchmark**).
*   **Yield**: Fluxo de caixa gerado pelo ativo (**Dividends**/Juros).
*   **Ripple ID**: Identificador único global para rastrear o efeito cascata.
*   **Time Machine**: Engine de reconstrução histórica e auditoria de estado.
