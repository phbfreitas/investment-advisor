2C. Auto-Calculated Fields

These fields are computed client-side from the holdings data — never stored in DB:

  • Weight % — holding market value / total portfolio value
  • P/L — (current price - book cost) × quantity
  • Market Value — current price × quantity (price from Yahoo)
  • Portfolio Totals — sum of book cost, market value, P/L across all holdings
  • Sector/Geo drift — actual allocation vs. target (feeds into existing drift table)

2D. Excel Export

Client-side CSV generation using native JavaScript (no library needed). Downloads as portfolio-YYYY-MM-DD.csv. Includes all visible columns plus a totals row. Opens natively in Excel.
